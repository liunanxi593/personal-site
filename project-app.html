Campus Second-hand Trading App

Background

This project is based on the campus second-hand trading context, where users need to browse, compare, and make decisions efficiently within a digital platform.

Problem

Second-hand trading platforms often involve scattered information, unclear trust cues, and inefficient user flows, which can make decision-making difficult for users.

Approach

This project designs the information architecture, user flow, and trust mechanisms for a campus second-hand trading platform. It also optimizes interface presentation and interaction feedback in order to support clearer navigation and better decision-making.
